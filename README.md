# Tic-Tac-Toe game

Web application made with React+Vite+TS for tic-tac-toe game

## Install

With [npm](https://npmjs.org/) installed, run
```
$ npm install
```

## Tech Stack

The following tools were used in the construction of the project:

#### **Platform** ([React](https://reactjs.org/) + [Vite](https://vite.dev/) + [TypeScript](https://www.typescriptlang.org/))

This config provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

- **[ESLint](https://github.com/eslint/eslint)**
- **[Typescript](https://github.com/microsoft/TypeScript/)**
- **[React router](https://reactrouter.com/home)**
